# Django Blog Project

Блог на Django с системой постов, тегов, комментариев и возможностью делиться статьями по email.

## 🌟 Основные возможности

- 📝 Публикация статей с тегами
- 💬 Система комментариев
- 📧 Поделиться статьей по email
- 🔖 Фильтрация по тегам

## 🛠 Технологии

- Django==5.2.3
- django-taggit==6.1.0
- Markdown==3.8
- python-dotenv==1.1.0
